# EduPlannerHub - منصة EduPlannerHub التعليمية

منصة تعليمية شاملة تربط الطلاب والمعلمين في مكان واحد، مصممة بالكامل باللغة العربية مع دعم كامل للتخطيط من اليمين لليسار (RTL).

## المميزات الرئيسية

### للطلاب:
- ✅ ملف شخصي قابل للتعديل
- ✅ رفع البحوث والواجبات والتقييمات الأسبوعية
- ✅ عرض الدرجات وملاحظات المعلمين
- ✅ الوصول إلى بنك الأسئلة الشامل
- ✅ الامتحانات التجريبية مع التصحيح التلقائي
- ✅ لوحة تحكم شخصية لمتابعة التقدم

### للمعلمين:
- ✅ ملف شخصي قابل للتعديل
- ✅ رفع الموارد التعليمية (PDFs، روابط، ملاحظات)
- ✅ إنشاء وإدارة بنك الأسئلة
- ✅ إنشاء وإدارة الامتحانات التجريبية
- ✅ إدارة الطلاب وتنظيمهم حسب الصفوف والمواد
- ✅ تقييم الطلاب وإعطاء الدرجات والملاحظات
- ✅ لوحة تحكم شاملة لمتابعة أداء الطلاب

## التقنيات المستخدمة

- **React 18** - مكتبة JavaScript لبناء واجهات المستخدم
- **Vite** - أداة بناء سريعة وحديثة
- **React Router** - للتوجيه بين الصفحات
- **Tailwind CSS** - لإطار عمل CSS سريع ومرن
- **Lucide React** - مكتبة الأيقونات
- **Google Fonts (Cairo & Tajawal)** - خطوط عربية جميلة

## متطلبات التشغيل

- Node.js (الإصدار 16 أو أحدث)
- npm أو yarn

## التثبيت والتشغيل

1. **تثبيت المكتبات:**
```bash
npm install
```

2. **تشغيل المشروع في وضع التطوير:**
```bash
npm run dev
```

3. **بناء المشروع للإنتاج:**
```bash
npm run build
```

4. **معاينة البناء:**
```bash
npm run preview
```

بعد تشغيل `npm run dev`، سيفتح المتصفح تلقائياً على `http://localhost:3000`

## التصميم والواجهة

- **التصميم:** حديث، نظيف، ومركز على التعليم
- **الألوان:** لوحة ألوان هادئة ومناسبة للتعليم
- **التخطيط:** دعم كامل للغة العربية من اليمين لليسار (RTL)
- **الاستجابة:** يعمل بشكل ممتاز على جميع الأجهزة (سطح المكتب، الجهاز اللوحي، الهاتف)

## الحالة الحالية

المنصة حالياً في وضع **عرض فقط** (View-Only Mode):
- لا يوجد نظام مصادقة بعد
- البيانات معروضة بشكل ثابت (Static Data)
- جميع الوظائف جاهزة للربط مع Backend في المستقبل
